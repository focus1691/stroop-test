# stroop-test
Gamified version of the famous Stroop Test experiment. Made with Phaser 3.

## Installation
`npm install`

## Scripts
- `npm start`

## The game
- **Part One** - Drag words to matching color.
- **Part Two** - Drag words to matching meaning.
